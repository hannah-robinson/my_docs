⬆ [[_Git_]]

Solution:
```bash
git reset HEAD~1 --soft
# resets commit but changes still persit in staging

git reset --mixed
# resets commit and unstages files

git reset --hard
# resets commit, unstages files and reverts the contents of the files too
```
